# popupdict.vim

Pop-up Japanese translation under a cursor.

![Screenshot](screenshot.png)

## Requirements

- Vim 8.1.1467

## Variables

| name              | description                     |
|---                |---                              |
| `g:popup_enabled` | `1` to enable pop-up dictionary |

## Dictionary sources

- https://github.com/kujirahand/EJDict
- https://github.com/monolithpl/verb.forms.dictionary
